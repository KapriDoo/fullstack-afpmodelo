Ejercicio 2
Descripción del caso
Imagina que se te ha pedido crear una aplicación web que permita a los usuarios buscar y ver información sobre películas utilizando la API pública de The Movie Database (TMDb).
Requerimientos
•	La aplicación debe tener una página principal que muestre una lista de las películas más populares actualmente en TMDb.
•	La aplicación debe tener una barra de búsqueda que permita a los usuarios buscar películas por título.
•	Cuando un usuario selecciona una película de la lista o de los resultados de búsqueda, se debe mostrar una página de detalles de la película que incluya: 
o	Título
o	Fecha de lanzamiento
o	Sinopsis
o	Poster de la película
o	Lista de actores principales
•	La aplicación debe tener un diseño atractivo y responsivo utilizando Bootstrap y CSS personalizado.
•	La aplicación debe estar construida utilizando componentes de Vue.js y Vuex para la gestión del estado.
API
Puedes utilizar la API pública de TMDb para obtener la información de las películas. Puedes encontrar más información sobre la API en su documentación.
Pauta de evaluación
Se evaluará la solución del candidato en base a los siguientes criterios:
•	Funcionalidad de la aplicación (30%)
•	Diseño y experiencia de usuario de la aplicación (30%)
•	Uso adecuado de las tecnologías propuestas (20%)
•	Claridad y organización del código (10%)
•	Creatividad y originalidad en la solución propuesta (10%)
Entrega
El candidato debe entregar el código fuente de la aplicación web en un repositorio público de GitHub. Además, debe proporcionar instrucciones claras y detalladas sobre cómo configurar y ejecutar el proyecto.
